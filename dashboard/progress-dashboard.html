Đánh Giá Kỹ Năng — Phan Duy Huy
⚙️ C# / .NET Core ⭐ Vững

3.5 năm thực tế tại HQSOFT. Đủ tốt để viết Clean Architecture. Lợi thế lớn nhất.

🗄️ SQL Server ⭐ Chuyên

Tối ưu query, execution plan — điểm mạnh nổi bật. Stored Procedures thành thạo.

🌐 ASP.NET Web API ⭐ Vững

Đã làm API đồng bộ dữ liệu đa hệ thống. RESTful design quen thuộc.

🏛️ Clean Architecture 🟡 Đang học

Đang áp dụng trong dự án. CQRS + Repository Pattern đã hoạt động tốt.

⚛️ ReactJS / TypeScript ❌ Cần học

Ưu tiên cao nhất. Frontend scaffold xong nhưng LoginPage/RegisterPage chưa có.

🔐 JWT + Refresh Token ✅ Hoàn thành

Đã implement đầy đủ: access token 15m, refresh token rotation, revoke endpoint.

⚡ Redis Caching 🟡 Cơ bản

Docker Redis đang chạy. RedisCacheService có nhưng chưa tích hợp vào features.

🐳 Docker / Compose 🟡 Trung bình

docker-compose.yml chạy SQL Server + Redis. Multi-container stack đang hoạt động.

🤖 Semantic Kernel / AI ⏳ Chưa học

Học sau Sprint 3 khi hệ thống base hoàn chỉnh. OpenAI embeddings + vector search.
